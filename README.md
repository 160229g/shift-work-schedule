# 교대근무 스케줄러

6인 교대근무팀을 위한 12주(84일) 근무표 자동 생성 도구입니다. 설치나 서버 없이 브라우저에서 파일만 열면 바로 사용할 수 있습니다.

## 사용법

1. **시작일 설정** — 스케줄을 시작할 날짜를 지정합니다.
2. **근무자명 입력** — 6명의 이름을 입력합니다.
3. **고정 휴무 지정** — 개인 사정으로 반드시 쉬어야 하는 날짜가 있으면 등록합니다. (해당 날짜 근무는 자동으로 다른 근무자에게 대체 배정됩니다)
4. **4주 단위 목표 휴무일수 입력** — 각 근무자가 4주마다 쉬어야 할 휴무일수를 정합니다.
5. **"스케줄 생성" 클릭** — 근무패턴을 바탕으로 12주치가 자동으로 채워집니다.
   - 필요하면 **구간 고정(잠금)** 체크박스로 특정 4주 구간을 잠가서, 이후 재생성해도 그 구간은 바뀌지 않게 할 수 있습니다.
6. **스케줄 표에서 직접 수정** — 셀을 클릭해 근무 유형을 바로 바꿀 수 있습니다.
7. **인원별 통계 확인** — 4주 단위 + 전체 합계로 형평성을 확인합니다.
8. **내보내기** — 엑셀(.xlsx), PNG, iCal(.ics), CSV 중 원하는 형식으로 다운로드합니다.
9. **저장/불러오기** — 작업 중인 스케줄을 JSON으로 저장했다가 나중에 다시 불러올 수 있습니다.

## 근무패턴 안내

기본 근무패턴은 **H(새벽) - S(오후) - N(숙직야간) - W(숙직새벽) - 휴무** 순서를 기본 골격으로 하되, 근무자의 휴가·고정휴무가 발생하면 다른 근무자를 대체근무자로 자동 배정하는 방식으로 구성되어 있습니다.

이 패턴은 한국산업안전보건공단의 「교대작업자의 보건관리지침」 중 '4. 작업관리 - 4.2. 교대작업자의 작업설계 시 고려 권장사항'을 참고해 설계했습니다.

<small>

> (1) 야간작업은 연속하여 3일을 넘기지 않도록 한다.
>
> (2) 야간반 근무를 모두 마친 후 아침반 근무에<br>
> 들어가기 전 최소 24시간 이상 휴식하도록 한다.
>
> (3) 주중보다는 주말에 쉬도록 하며,<br>
> 하루씩 띄어 쉬는 것보다 주말에 이틀 연이어 쉬도록 한다.
>
> (5) 근무반 교대방향은 아침반 → 저녁반 → 야간반 순으로<br>
> 정방향 순환이 되게 한다.

</small>

## 자동으로 지켜지는 규칙

- 월~토요일은 핵심 근무(S/N/W/H)에 항상 4명 이상 배정
- 고정 휴무 지정 시 대체 근무자 자동 배정
