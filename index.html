교대근무 스케줄러

6인 · 12주(84일) 생성 · 제공된 근무패턴(인원별 84일) 그대로 배정

1기본 설정

2근무자 이름

3규칙 요약

· [기본 생성 로직] 제공된 근무패턴(인원별 84일)을 가공 없이 그대로 배정합니다.
· [휴무 픽스 적용] 고정된 휴무일의 원래 근무는 당일 근무인 S근무자 중 1명에게 자동으로 대직 배정됩니다.
· [목표 미달 시] 4주 목표 휴무일 미달 시 평일(월~금) S근무만 휴무(O)로 전환합니다.
· [인원 유지] 월~토요일은 S, N, W, H 근무자가 최소 4명 이상이어야 합니다(일요일 제외).

4휴무 픽스 (인원별 최대 3일)

지정한 날짜는 해당 근무자가 무조건 휴무로 고정되며, 원래 근무는 당일 S근무자가 대직합니다.

4.54주 단위 휴무 목표치 설정

각 4주 기간별 목표 휴무일 수입니다. 미달 시 S근무를 휴무(O)로 전환합니다.
근무자1-4주 목표5-8주 목표9-12주 목표
체크된 구간은 "스케줄 생성"을 다시 눌러도 현재 표시된 값(수동 수정분 포함) 그대로 유지되고, 해당 구간의 픽스 휴무/목표휴무 재계산도 건너뜁니다. 시작일을 바꾸면 날짜가 어긋나므로 고정이 자동 해제됩니다.

5스케줄

6인원별 통계 (형평성 확인, 4주 단위)

7내보내기

4주 단위로 3개 시트(1~4주 / 5~8주 / 9~12주)가 포함된 엑셀 파일이 생성됩니다.

8스케줄 저장 / 불러오기

현재 설정(근무자명·휴무픽스·목표·생성된 스케줄) 전체를 파일로 저장하거나, 저장했던 파일을 다시 불러올 수 있습니다.
